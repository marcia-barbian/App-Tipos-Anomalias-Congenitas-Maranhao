## O repositório App-Anomalias-Congenitas-Maranhao

**App-Anomalias-Congenitas-Maranhao** é um pacote do **R** contendo um aplicativo em [shiny](https://shiny.rstudio.com) que produz gráficos detalhados e interativos de casos de anomalias congênitas no estado do Maranhão entre os anos de 2010 e 2016. Desenvolvemos o aplicativo para fornecer aos pesquisadores saídas gráficas detalhadas geograficamente, por meio do uso de um software com interface amigável, podendo, portanto, ser facilmente utilizado por quem não conhece uma linguagem de programação. Através do App-Anomalias-Congenitas-Maranhao o usuário terá a possibilidade de exibir informações sobre a prevalência de casos de anomalias congênitas por município no estado do Maranhão, em que é possível escolher o grupo de anomalias a ser considerado na geração dos mapas.
